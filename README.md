# FCMS - Fast Content Management System
FCMS is a content management system built with lumen. This CMS is built with the aim of being a simple and fast CMS.

## What features are on the roadmap?

1. [ ] Content management
2. [ ] User management
3. [ ] Media management
4. [ ] Custom pages management
5. [ ] Security management
6. [ ] SEO management
7. [ ] Blog management
8. [ ] REST API

## How to install?
1. This app is built with lumen 10.x, see the [lumen documentation](https://lumen.laravel.com/docs/10.x#installation) for more information.
2. Clone this repository
3. Run `composer install`
4. Run `php artisan fresh:install`
5. Run `php artisan serve` or if you use laragon, just go to the browser and type `fcms.test`. See the [laragon documentation](https://laragon.org/docs/pretty-urls.html) for more information.
